This is a template works well for development tasks as well as general business workflows. It's great for drafting requirements. I reference it all the time.
<br/>

User Story: As a [....User Role, such as Property Manager, etc....], I need the ability to [.....action / workflow needed.....], so I can [....benefit this feature will add....]
<br/>

Example: As a Property Manager, I need the ability to flag a lease as non-renewable so that I can easily keep track of which leases we don't want to offer renewals to.
<br/>

As an analyst, I need the ability to communicate susinctly, so I can translate business needs into technical requirements.
